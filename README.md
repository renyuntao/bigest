## SYNOPSIS

```python
$ bigest /data
$ bigest 
```

## DESCRIPTION

Find the biggest file/directory under a path

## Example

```python
$ bigest /home/renyuntao/
    30.7G  /home/renyuntao/VirtualBox VMs
    16.5G  /home/renyuntao/VirtualBox VMs/Win7
    16.5G  /home/renyuntao/VirtualBox VMs/Win7/Win7.vdi
```
